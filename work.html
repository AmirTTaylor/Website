Tic Tac Toe

This is a simple yet interactive Tic-Tac-Toe game developed using JavaFX. It allows two players to take turns marking "X" or "O" on a 3x3 grid. The game includes functionality to check for a win or draw after each move and displays the result accordingly. Players can restart the game at any time using the "Play Again" button, which appears once a winner is determined or the game results in a draw.
